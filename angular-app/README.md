# Angular App

Angular setup built with [Angular CLI](https://angular.dev/tools/cli).

## Development

Start the local dev server:

```bash
ng serve
```

Open `http://localhost:4200/` in your browser. The app reloads automatically on source file changes.

## Build

```bash
ng build
```

Compiled output is generated in the `dist/` directory, optimized for production by default.

## Testing

Run unit tests (via [Vitest](https://vitest.dev/)):

```bash
ng test
```
